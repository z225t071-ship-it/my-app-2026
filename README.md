# 📄 お役所書類のバカよけ翻訳機

「お役所の言葉は、なぜこんなに難しいのか？」  
そんな悩みを解決するために生まれた、行政文書を「誰にでもわかる言葉」に言い換えるシンプルな翻訳ツールです。

**URL:** [https://z225t071-ship-it.github.io/my-app-2026/](https://z225t071-ship-it.github.io/my-app-2026/)

## 🚀 使い方

1. **入力する**  
   左側のテキストエリアに、役所から届いた通知や書類の難しい文章をコピー＆ペーストします。
2. **翻訳ボタンを押す**  
   「翻訳（やさしく言い換え）」ボタンをクリックします。
3. **結果を確認・コピー**  
   右側に表示された「やさしい言葉」を確認します。翻訳された箇所は青色で強調されます。必要に応じて「結果をコピー」ボタンでクリップボードに保存できます。

## ✨ 主な機能

- **辞書ベースの即時変換**: 40種類以上のお役所特有の言い回し（「可及的速やかに」「遺漏なく」等）に対応。
- **サンプル機能**: 「サンプルを入力」ボタンで、どのような変換が行われるかすぐに試せます。
- **レスポンシブデザイン**: PCだけでなく、スマートフォンからも快適に利用可能です。
- **Official & Clean UI**: 役所の書類を扱う際の緊張感を和らげる、清潔感のあるデザインを採用。

## 📖 変換例

- **可及的速やかに** → できるだけ早く
- **遺漏なく** → 漏れがないようにしっかり
- **〜されたい** → 〜してください
- **ご査収ください** → 内容を確認してください

## 🛠 技術構成

- **HTML5 / CSS3**: Vanilla CSSによるモダンなレイアウト
- **JavaScript**: フレームワークなし（Vanilla JS）で軽量に動作
- **GitHub Pages**: 静的ホスティングによる高速な公開
