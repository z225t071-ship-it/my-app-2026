## 🚀 使い方 / How to Use

- **[JP]** テキストを入力するだけ。リアルタイムで「やさしい言葉」に変換されます。
- **[CN]** 只需输入文本。它将实时转换为易于理解的语言。
- **[KR]** 텍스트를 입력하기만 하면 됩니다. 실시간으로 이해하기 쉬운 말로 변환됩니다.
- **[EN]** Just type. It converts bureaucratic terms to plain language in real-time.
